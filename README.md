# JavaScript - Estruturas Condicionais

Projeto desenvolvido durante os estudos de **JavaScript**, com foco na prática de estruturas condicionais e lógica de programação por meio de desafios propostos em curso.

Os exercícios simulam situações do cotidiano para reforçar a tomada de decisão utilizando `if`, `else if` e `else`, além da interação com o usuário através de `prompt()` e `alert()`.

---

## Sobre o projeto:

Este projeto reúne uma série de desafios desenvolvidos para consolidar os fundamentos da linguagem JavaScript, praticando validações, operadores lógicos e relacionais, além do controle do fluxo da aplicação.

Embora os exercícios sejam independentes, todos possuem o objetivo de fortalecer a capacidade de resolver problemas utilizando lógica de programação.

---

## Desafios implementados:

### Verificação do dia da semana

Solicita ao usuário o dia da semana e informa se é dia útil ou fim de semana.

Além do desafio original, foi implementada uma validação para impedir a entrada de dias inexistentes.


### Identificação de números

Recebe um número informado pelo usuário e identifica se ele é:

- Positivo;
- Negativo;
- Igual a zero.

Também trata entradas inválidas.


### Sistema de pontuação

Simula uma pontuação de jogo.

Caso a pontuação seja maior ou igual a **100 pontos**, o jogador vence. Caso contrário, é incentivado a tentar novamente.


### Consulta de saldo

Exibe o saldo da conta utilizando **Template Strings** e formatação monetária com `toFixed(2)`.


### Saudação personalizada

Solicita o nome do usuário e apresenta uma mensagem de boas-vindas personalizada.
